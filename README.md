# Flight Delay Prediction using Weather Data (AWS)

This project designs and implements a cloud-based data pipeline on AWS
to analyze and predict flight delays using historical flight data and
real-time weather information.

## Technologies
- AWS S3
- AWS Lambda
- AWS Glue
- AWS Athena
- OpenWeather API

## Business Goal
Reduce operational costs and improve passenger experience by identifying
high-risk delay conditions in advance.
